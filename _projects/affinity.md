## Overview

Affinity is a relationship analysis product designed to help individuals understand their dating behaviour through structured self-reflection and data analysis.

## Problem

Most people approach dating without a systematic understanding of their own preferences, patterns, and decision-making biases. This leads to repeated mistakes, misaligned expectations, and frustration.

## Solution

Affinity provides a structured framework for capturing relationship data, identifying behavioural patterns, and generating actionable insights. It combines behavioural science with practical tooling to bring clarity to an inherently complex domain.

## Architecture

The system is built as a web application with a Python backend handling data processing and a React frontend for user interaction. Data is stored in PostgreSQL and processed through custom analytical pipelines.

## Demo

A working prototype is currently in development. The system allows users to input relationship data, run analysis routines, and view pattern reports.
